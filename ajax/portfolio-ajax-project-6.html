Python RPG Game

Project Info:

Select a class and help save the ATL tech village from the Python by first taking down his developer squad. They drop currency (rubber duckies) which can be used to purchase items in the shop and upgrade your character. Make sure your character is upgraded enough before facing the Python or face defeat! Code makes use of: Import functions, Functions, While Loops, Classes, Variables Game can be found in the rpgClass.py file.
