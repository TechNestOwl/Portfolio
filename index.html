Technologies I Use

Javascript

JavaScript is a dynamic programming language that supports Math calculations, it allows me to dynamically add HTML content to the DOM, create dynamic style declarations, fetch content from other websites, and lots more.

React

React is one of the more popular front-end JavaScript libraries. React brings many advantages to the table, for example, React allows one-way data binding and encourages the practice of DOM manipulation. It also uses lifecycle methods to execute code at set points in case of component failure.

Redux.js

Redux is a javascript library used for managing application state. Redux creates a process and procedures to interact with the store so that components will not just update or read the store randomly.

PostgresSQL

PostgreSQL is an object-relational database management system. It supports a large part of the SQL standard and offers many modern features like complex queries and foreign keys.

Node.js

Node.js is a back-end JavaScript runtime environment. Some of it's benefits are that it uses an event-driven, input/output model that makes it lightweight and efficient. Also, it's package ecosystem (npm) is massive.

AWS

AWS is amazon's implementation of cloud services, it offers a lot of services and technologies that make life easier.

HTML

Simply put, this is a markup langauge I use to design and display web page structure.

CSS

This is a simple design sheet language used to handle the look and feel of a web page. In short, CSS provides powerful control over the presentation of an HTML document.
